FROST-configManager:

Program to easly change configuration of the FROST-OPI-ZERO temperature monitoring setup.

--> Tested with winscp 5.1.11 installed on pc.
--> Uses Winscp to write and read configuration files.
--> Login system uses linux ssh login. No passwords are saved in the code.
--> When using discovery, use you linux default login credentials. Beware, discovery only set for 192.168.1.xxx networks. Discovery limited to 192.168.1.1 until 192.168.1.30
